# Что?
Бот который предоставляет информацию о вакансиях в Минске

# Какой функционал?
1. Меню кнопок
2. Выбор професси
3. Выбор сколько получать вакансий в ответе
4. Выбор сортировки либо по дате либо по зп
5. Возможность просмотра вакансий
6. Возможность просмотра предидущих запросов
7. Сохранение вакансии в закладки

# Как взаимодействовать?
/start - Приветсвие
